Discrete Gaussian Samplers over Lattices#

This file implements oracles which return samples from a lattice following a discrete Gaussian distribution. That is, if \(Ïƒ\) is big enough relative to the provided basis, then vectors are returned with a probability proportional to \(\exp(-|x-c|_2^2/(2Ïƒ^2))\). More precisely lattice vectors in \(x âˆˆ Î›\) are returned with probability:

\(\exp(-|x-c|_2^2/(2ÏƒÂ²))/(âˆ‘_{x âˆˆ Î›} \exp(-|x|_2^2/(2ÏƒÂ²)))\)

__init__(B, sigma=1, c=None, precision=None)#

Construct a discrete Gaussian sampler over the lattice \(Î›(B)\) with parameter sigma and center \(c\).

INPUT:

  • B â€“ a basis for the lattice, one of the following:

    • an integer matrix,

    • an object with a matrix() method, e.g. ZZ^n, or

    • an object where matrix(B) succeeds, e.g. a list of vectors.

  • sigma â€“ Gaussian parameter \(Ïƒ>0\).

  • c â€“ center \(c\), any vector in \(\ZZ^n\) is supported, but \(c âˆˆ Î›(B)\) is faster.

  • precision â€“ bit precision \(â‰¥ 53\).
