sphinxcontrib_applehelp: Sphinx extension which outputs Apple help book
=======================================================================

Description
-----------

Sphinx extension which outputs Apple help book

License
-------

BSD

Type
----

standard


Dependencies
------------

- $(PYTHON)
- $(PYTHON_TOOLCHAIN)

Version Information
-------------------

    1.0.8

install-requires.txt::

    sphinxcontrib_applehelp >=1.0.2
