.. _spkg_cypari:

cypari2: Python interface to the number theory library libpari
==============================================================

Description
-----------

A Python interface to the number theory library libpari.

License
-------

GPL version 2 or later


Upstream Contact
----------------

https://github.com/defeo/cypari2

Type
----

standard


Dependencies
------------

- $(PYTHON)
- :ref:`spkg_cython`
- :ref:`spkg_pari`
- :ref:`spkg_cysignals`
- $(PYTHON_TOOLCHAIN)

Version Information
-------------------

package-version.txt::

    2.1.3

install-requires.txt::

    cypari2 >=2.1.1


Equivalent System Packages
--------------------------

conda:

.. CODE-BLOCK:: bash

    $ conda install  cypari2


See https://repology.org/project/python:cypari2/versions


However, these system packages will not be used for building Sage
because using Python site-packages is not supported by the Sage distribution;
see :issue:`29023`
