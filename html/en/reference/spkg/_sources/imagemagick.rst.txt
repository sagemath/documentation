Version Information

Equivalent System Packages
--------------------------

alpine:
install the following packages: imagemagick

arch:

.. CODE-BLOCK:: bash

    $ sudo pacman -S  imagemagick


conda:

.. CODE-BLOCK:: bash

    $ conda install imagemagick


cygwin:

.. CODE-BLOCK:: bash

    $ apt-cyg install ImageMagick


Debian/Ubuntu:

.. CODE-BLOCK:: bash

    $ sudo apt-get install  imagemagick


Fedora/Redhat/CentOS:

.. CODE-BLOCK:: bash

    $ sudo yum install  ImageMagick


freebsd:

.. CODE-BLOCK:: bash

    $ sudo pkg install graphics/ImageMagick7


homebrew:

.. CODE-BLOCK:: bash

    $ brew install imagemagick


macports:
install the following packages: ImageMagick

nix:

.. CODE-BLOCK:: bash

    $ nix-env --install imagemagick


opensuse:

.. CODE-BLOCK:: bash

    $ sudo zypper install ImageMagick


void:

.. CODE-BLOCK:: bash

    $ sudo xbps-install  ImageMagick


See https://repology.org/project/imagemagick/versions


If the system package is installed, ./configure will check whether it can be used.
