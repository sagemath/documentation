.. _spkg_4ti2:

4ti2: Algebraic, geometric and combinatorial problems on linear spaces
======================================================================

Description
-----------

A software package for algebraic, geometric and combinatorial problems
on linear spaces. Available at https://4ti2.github.io/.

License
-------

Equivalent System Packages
--------------------------

.. tab:: Arch Linux

   .. CODE-BLOCK:: bash

       $ sudo pacman -S 4ti2 


.. tab:: conda-forge

   .. CODE-BLOCK:: bash

       $ conda install 4ti2 


.. tab:: Cygwin

   .. CODE-BLOCK:: bash

       $ apt-cyg install lib4ti2_0 lib4ti2-devel 


.. tab:: Debian/Ubuntu

   .. CODE-BLOCK:: bash

       $ sudo apt-get install 4ti2 


.. tab:: Fedora/Redhat/CentOS

   .. CODE-BLOCK:: bash

       $ sudo yum install 4ti2 


.. tab:: FreeBSD

   .. CODE-BLOCK:: bash

       $ sudo pkg install math/4ti2 


.. tab:: Gentoo Linux

   .. CODE-BLOCK:: bash

       $ sudo emerge sci-mathematics/4ti2 


.. tab:: openSUSE

   .. CODE-BLOCK:: bash

       $ sudo zypper install 4ti2 4ti2-devel
