Version Information
-------------------

package-version.txt::

    3.27.0

Equivalent System Packages
--------------------------

arch:

.. CODE-BLOCK:: bash

    $ sudo pacman -S  python-tox


conda:

.. CODE-BLOCK:: bash

    $ conda install  tox


Debian/Ubuntu:

.. CODE-BLOCK:: bash

    $ sudo apt-get install  tox


Fedora/Redhat/CentOS:

.. CODE-BLOCK:: bash

    $ sudo yum install  tox


freebsd:

.. CODE-BLOCK:: bash

    $ sudo pkg install tox


gentoo:

.. CODE-BLOCK:: bash

    $ sudo emerge dev-python/tox


homebrew:

.. CODE-BLOCK:: bash

    $ brew install tox


macports:
install the following packages: py-tox

slackware:

.. CODE-BLOCK:: bash

    $ sudo slackpkg install tox


void:

.. CODE-BLOCK:: bash

    $ sudo xbps-install  tox


See https://repology.org/project/python:tox/versions


If the system package is installed, ./configure will check whether it can be used.
