gmp: Library for arbitrary precision arithmetic
===============================================

Description
-----------

GMP is a free library for arbitrary precision arithmetic, operating on
signed integers, rational numbers, and floating-point numbers. There is
no practical limit to the precision except the ones implied by the
available memory in the machine GMP runs on. GMP has a rich set of
functions, and the functions have a regular interface.

Equivalent System Packages
--------------------------

conda:

.. CODE-BLOCK:: bash

    $ conda install gmp

cygwin:

.. CODE-BLOCK:: bash

    $ apt-cyg install libgmp-devel

Debian/Ubuntu:

.. CODE-BLOCK:: bash

    $ sudo apt-get install  libgmp-dev

Fedora/Redhat/CentOS:

.. CODE-BLOCK:: bash

    $ sudo yum install  gmp gmp-devel

freebsd:

.. CODE-BLOCK:: bash

    $ sudo pkg install math/gmp

gentoo:

.. CODE-BLOCK:: bash

    $ sudo emerge dev-libs/gmp

homebrew:

.. CODE-BLOCK:: bash

    $ brew install gmp

macports:
install the following packages: gmp
opensuse:

.. CODE-BLOCK:: bash

    $ sudo zypper install gmp-devel

slackware:

.. CODE-BLOCK:: bash

    $ sudo slackpkg install gmp

void:

.. CODE-BLOCK:: bash

    $ sudo xbps-install  gmpxx-devel
