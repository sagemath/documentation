tinycss2: A tiny CSS parser
===========================

Description
-----------

A tiny CSS parser

License
-------

Upstream Contact
----------------

https://pypi.org/project/tinycss2/


Type
----

standard


Dependencies
------------

- :ref:`spkg_webencodings`
- $(PYTHON_TOOLCHAIN)
- $(PYTHON)

Version Information
-------------------

package-version.txt::

    1.2.1

install-requires.txt::

    tinycss2


Equivalent System Packages
--------------------------

alpine:

.. CODE-BLOCK:: bash

    $ apk add py3-tinycss2


arch:

.. CODE-BLOCK:: bash

    $ sudo pacman -S  python-tinycss2


conda:

.. CODE-BLOCK:: bash

    $ conda install  tinycss2


Debian/Ubuntu:

.. CODE-BLOCK:: bash

    $ sudo apt-get install  python-tinycss2


Fedora/Redhat/CentOS:

.. CODE-BLOCK:: bash

    $ sudo yum install  python-tinycss2


freebsd:

.. CODE-BLOCK:: bash

    $ sudo pkg install textproc/py-tinycss2


gentoo:

.. CODE-BLOCK:: bash

    $ sudo emerge dev-python/tinycss2


macports:
install the following packages: py-tinycss2

opensuse:

.. CODE-BLOCK:: bash

    $ sudo zypper install python-tinycss2


void:

.. CODE-BLOCK:: bash

    $ sudo xbps-install  python3-tinycss2



If the system package is installed and if the (experimental) option
--enable-system-site-packages is passed to ./configure, then ./configure
will check if the system package can be used.
