.. _spkg_cliquer:

cliquer: Routines for clique searching
======================================

Description
-----------

Cliquer is a set of C routines for finding cliques in an arbitrary
weighted graph. It uses an exact branch-and-bound algorithm
developed by Patric Östergård.
